MIDI Documents

For an initial orientation to MIDI (and also some geeky details), it's suggested that you read MIDI, an introductory essay by Christopher Dobrian.

The following examples in the Max Cookbook will help you learn how to use MIDI in Max.

  • Useful MIDI Objects
    - doesn't do anything, but shows available objects for handling MIDI data
  • MIDI mapping to amplitude
    - using MIDI to control the volume of a sound
