Resources and Links

Since JavaScript is one of the most popular languages in the world, we are not going to provide our own intro-to-JavaScript tutorial, as to avoid NIH syndrome. Instead, we expect you to learn the language on your own, from various tutorials and/or books linked here. While this may seem like a tall order, you'll find that learning your second programming language will be much easier than learning your first, not to mention that the ability to quickly learn and translate between languages will be essential when working in the real world!

To provide you with some guidance and to keep you on track, we have created a set of questions for understanding at the end of this module: if you feel comfortable answering each of those questions, you are ready to move on to the more advanced parts of this course.

Finally, this module contains a short primer describing JavaScript syntax, as well as a number of videos summarizing the differences between JavaScript and other languages you may know. (Here, here, here, and here.) You may want to start with the primer and these videos, and if they seem too confusing or over-your-head, then you might benefit from spending some time with a few of the resources provided below.

Resources to Learn JavaScript

  • Best-Of Book List
  • Best-Of Online Tutorial List
  • Codecademy
    • Codecademy has some free resources, some paid; the interactive element is what sets it apart from other tutorials.
  • The Mozilla Developer Network tutorials
    • The MDN documentation should be your go-to when trying to answer JavaScript syntax and library questions, and their tutorials are clear and comprehensive.
  • JavaScript: The Good Parts by Douglas Crockford
    • An oldie-but-goodie, the Crockford summarizes some of the growing pains JavaScript has undergone the past several years, while also containing some good philosophy when it comes to good programming vs. bad programming.

Resources to Learn HTML

  • w3schools

Other Resources

  • Stack Overflow
    • Absolutely the best programming Q&A site.
  • Google's JavaScript Style Guide or AirBnB's JavaScript Style Guide
    • Two of many style guides: it's less important which style guide you follow, more that you just follow one at all.

Above and Beyond

The following subjects are not strictly-speaking relevant for this course, but if you can surmount the additional learning curves, you'll find that your web-based projects will be far simpler and far more elegant.

  • Learning jQuery will make communication between JavaScript and HTML 1000 times easier. There are jQuery tutorials on Codecademy and on the jQuery site.
  • If you are building a large web application, chances are at some point you'll be exposed to one of many JavaScript frameworks. Among the most popualar are Angular, React, Ember, and Backbone; each of those sites have their own tutorials.
  • Learning CSS is vital if you want to make your web applications actually look good. There exist CSS frameworks like Bootstrap and Foundation, as well as more low-level extensions of CSS like Sass and Less. Usually, CSS is taught along with JavaScript and HTML as a sort of HTML/CSS/JavaScript-hybrid, but since there is so much else to learn in this course, we are making the CSS portion optional.
